## ğŸ“¦ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/indie-hacker-toolkit.git

# Navigate to the project
cd indie-hacker-toolkit

# Install dependencies
npm install

# Run development server
```
